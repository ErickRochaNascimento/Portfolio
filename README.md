# Portfolio Pessoal

Este repositório contém o código-fonte do meu **Portfólio Pessoal**, uma página web desenvolvida para apresentar meus projetos, habilidades e informações de contato.

## 🎯 Objetivo

Servir como um cartão de visitas digital, permitindo que recrutadores, colegas e interessados possam visualizar meu trabalho e trajetória profissional de forma organizada.

## 💡 Funcionalidades

*   **Design Responsivo:** O layout se adapta a diferentes tamanhos de tela (desktop, tablet e mobile).
*   **Seções Chave:** Inclui seções como "Sobre Mim", "Habilidades", "Projetos" e "Contato".
*   **Links Diretos:** Fornece links diretos para os repositórios dos projetos em destaque.

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estrutura semântica da página.
*   **CSS3:** Estilização e design.
*   **JavaScript:** Para interatividade e possíveis animações.
